# agricola_pluviometro
projeto com objetivo de realizar leituras de solo e chuva para apoiar agricultores, focado no Objetivo de Desenvolvimento Sustentável 2 "Fome Zero e Agricultura Sustentável", visa acabar com a fome, garantir o acesso a alimentos seguros, nutritivos e suficientes para todos, especialmente os mais pobres e vulneráveis, até 2030
